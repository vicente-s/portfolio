Hi! I'm Vicente Soriano, a software developer born and raised in New York City - go Knicks! I became a developer because I wanted to create my ideas. Eventually, I was able to help others create theirs too and it has been an amazing journey. I also enjoy skateboarding, video games, and anything starring Dave Chappelle.

I'm a UX-driven problem-solver with a passion for design and a solid background in building applications. Proficient with JavaScript, React, Python, Django, Ruby, Ruby on Rails, SQL, PostgreSQL, Git, HTML, CSS, Bootstrap, Swift, jQuery, Sketch, REST APIs, MVC Patterns, Postman, and anything else that gets the job done.
